Course Description

Machine Learning introduces students to the concepts, algorithms and practical applications of intelligent systems. The course focuses on supervised learning, unsupervised learning, classification, regression, clustering and model evaluation techniques. Students gain practical experience using Python and popular machine learning libraries to build predictive models for real-world problems.

Learning Outcomes

  • Understand the fundamentals of Machine Learning.
  • Develop predictive models using Python.
  • Apply classification and regression algorithms.
  • Perform clustering and data preprocessing.
  • Evaluate model performance using standard metrics.
  • Build real-world AI and Machine Learning applications.
